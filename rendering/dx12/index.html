3DC++GPU学校项目流体模拟

DX12

3D, C++ and DX12 based Dialogue playable game

课程描述

讲述基本图形学数学、光栅化流程、基本GPU概念、GPU管线、DX12接口、几何、动画、纹理采样、游戏图像、碰撞

项目描述

自行利用C++与DX12库构建DX12渲染管线，要求构建、渲染一个游戏场景。其中包括数据驱动型场景加载，具有纹理的几何，需要透明度测试的网格体，具有顶点位移动画的网格体，实例化构建的网格体。 具有动画以及动画融合过度的网格体，第一人称的相机移动，简单碰撞，法向映射，特殊效果(天空盒、粒子系统、水体）

结果

DX12管线C++配置，shader，场景，Gerstner wave模拟(材质有点塑料)，简单PCG树摆放，相机运动等等

游戏概念为名为“星际穿越之后”的游戏关卡，主要讲述了库珀（电影《星际穿越》的主角）在进入黑洞后遇到的景象。

这里有实例化的树木，一片受黑洞影响的海洋，以及一个可以与你对话的4D生物（霸王龙，它救了库珀）。玩家以库珀的第一人称视角，只能向前移动（受碰撞限制），与霸王龙对话，然后离开。
